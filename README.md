# Ayur Vidya Nikethan (AVN) — Centre for Integrated Wellness

A premium single-page website for **Ayur Vidya Nikethan (AVN)**, a yoga &amp; wellness institution
offering structured yogic wellness programs for busy, high-responsibility professionals.

> *Where Wellness Becomes A Way Of Life.*

## Sections

- Hero with free-session video + trust strip
- The Modern Professional Wellness Crisis
- Signature Programs™ (Stress Reset, Desk Work Recovery, Sleep &amp; Burnout, Complete Transformation)
- About AVN / Welcome
- Who We Serve
- Why Professionals Choose AVN + Complimentary Session CTA
- Daily Live Program Schedule (IST) + flexible global access
- Meet Our Experts (Yogacharya Lijoy Raj C.R. &amp; Mrs. Leema Prasad)
- Testimonials / Success Stories
- Program Investment / Pricing
- Book Free Wellness Consultation + Enquiry form
- FAQ (accordion)
- Footer

Built on Bootstrap 5 + Vite + SCSS. Imagery sourced from [Pexels](https://www.pexels.com/).

## Getting Started

1. Install dependencies
```
npm i
```
2. Run the development server
```
npm run dev
```
3. Build for production
```
npm run build
```
